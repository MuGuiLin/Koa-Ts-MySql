# Node.js Koa2 + TypeScript + MySql 项目实战

#### 项目目录介绍：

**backend 后端代码目录**

启动方式：

```shell 进入backend目录，在当前目录打开命令行工具：
1、进入backend目录
cd backend

2、在当前目录打开命令行工具，输入如下命令
npm start
或
yarn start
```



**frontend 前端代码目录**

启动方式：

```shell
1、进入frontend目录
cd frontend

2、在当前目录打开命令行工具，输入如下命令
npm run serve
或
yarn serve
```
